Construct vector search queries

Just as you did in the previous step, you can explore the search results using an aggregation pipeline.

Querying with the aggregation builderâ€‹

Using the aggregation builder, or the MongoDB Shell, you can construct a pipeline that uses the $search stage to query the vectorsearch index.

Add the following code for the $search stage.

{
    index: "vectorsearch",
    knnBeta: {
        vector: `vector goes here`,
        path: "plot_embedding",
        k: 20
    }
}

The stage uses the vectorsearch index. Rather than using the 'text' operator as we did for full-text searches, we are using the knnBeta operator this time.
